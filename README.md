# 💚 Hacker Login Page

A stunning login page with **Green & Teal hacker aesthetic** built with pure **HTML & CSS** — featuring space galaxy background, matrix falling code, glassmorphism card, and smooth glow animations.

![Login Page Preview](preview.png)

---

## 🚀 Live Demo

Open `index.html` directly in your browser — no setup needed!

---

## ✨ Features

- 🌌 **Space galaxy background** — deep dark universe vibes
- 💚 **Green & Teal hacker theme** — full terminal aesthetic
- 👾 **Matrix falling code** — `01 10 </> {}` animation in background
- ✨ **Twinkling stars** — subtle space details
- 🔶 **Green glow focus effect** — inputs glow on click
- 🎞️ **Slide-up animation** on page load
- 🖱️ **Shimmer hover effect** on login button
- 📱 **Fully responsive** — works on mobile and desktop
- 💯 **Pure HTML & CSS** — zero JavaScript, zero libraries

---

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox, Grid, Backdrop Filter, Keyframe Animations)
- Google Fonts (Syne + Share Tech Mono + DM Sans)
